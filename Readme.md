# Personal Portfolio Website

A full stack personal portfolio website built with HTML, CSS, JavaScript, Node.js and MongoDB.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## How to Run Locally

### 1. Clone the repo
git clone https://github.com/samimali832925-eng/portfolio.git
cd portfolio

### 2. Install dependencies
npm install

### 3. Create a .env file in the root folder
MONGO_URI=mongodb://localhost:27017/portfolio
PORT=5000

### 4. Make sure MongoDB is running
Open MongoDB Compass and connect to localhost:27017

### 5. Start the server
npm run dev

### 6. Open in browser
http://localhost:5000

## Folder Structure
- client/ — Frontend (HTML, CSS, JS)
- server/ — Backend (Node.js, Express, MongoDB)
